# Jozi Nites Blog

A sex-positive blog for the Black community in Johannesburg. Built for GitHub Pages.

## How to Host on GitHub

1. Create a new repository called `yourusername.github.io`
2. Upload all these files to the repo
3. Go to Settings → Pages → Select main branch
4. Your site will be live at https://yourusername.github.io

Features included:
- Dark + Gold Jozi Nites theme
- Social login simulation
- Comment-ready structure (add Giscus later)
- Full tag list from your request
- All images feature Black characters

Made with love for Johnny Boy & the culture. 🖤✨
